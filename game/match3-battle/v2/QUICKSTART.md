# 🎮 快速开始指南

欢迎使用多人实时对战消消乐 App！这是一个 5 分钟快速启动指南。

## 📦 第一步：解压文件

将下载的 `match3-battle-app.zip` 解压到任意文件夹。

## ⚙️ 第二步：安装 Node.js (如果还没有)

访问 https://nodejs.org/ 下载并安装 LTS 版本。

## 🚀 第三步：启动服务器

### Windows 用户
双击运行 `start.bat`

### Mac/Linux 用户
```bash
./start.sh
```

或者手动启动：
```bash
cd server
npm install    # 首次运行需要
npm start
```

## 🎯 第四步：开始游戏

1. 打开浏览器访问：`http://localhost:3000`
2. 选择游戏模式：
   - **创建房间**：生成房间号，分享给朋友
   - **加入房间**：输入 6 位房间号加入
   - **随机匹配**：自动匹配在线对手

## 📱 手机访问

1. 确保手机和电脑在同一 WiFi
2. 在电脑上运行 `ipconfig`(Windows) 或 `ifconfig`(Mac/Linux) 查看 IP
3. 手机浏览器访问：`http://你的IP:3000`

## 🎮 游戏玩法

1. **选择宝石**：点击一个宝石
2. **交换宝石**：点击相邻的宝石进行交换
3. **消除得分**：横向或纵向 3 个相同颜色即可消除
4. **限时对战**：60 秒倒计时，分高者获胜

## 💬 聊天功能

点击右下角 💬 按钮打开聊天面板，可以发送文字和表情。

## ❓ 遇到问题？

查看详细文档：
- `INSTALLATION.md` - 完整安装指南
- `README.md` - 项目说明文档

## 🎉 就这么简单！

现在开始享受游戏吧！祝你玩得开心！

---

**提示**：如果服务器启动失败，可能是端口被占用。可以在 `server/server.js` 中修改端口号。
