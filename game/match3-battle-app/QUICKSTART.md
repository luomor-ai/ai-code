# 🚀 快速开始指南

## 方式一：使用启动脚本（推荐）

### Windows 用户
1. 双击 `start.bat` 文件
2. 等待依赖安装完成
3. 看到 "服务器已启动" 提示后，打开浏览器访问 `http://localhost:3000`

### Mac/Linux 用户
1. 打开终端，进入项目目录
2. 运行 `./start.sh`
3. 看到 "服务器已启动" 提示后，打开浏览器访问 `http://localhost:3000`

## 方式二：手动启动

### 1. 安装依赖
```bash
cd server
npm install
```

### 2. 启动服务器
```bash
npm start
```

### 3. 访问游戏
在浏览器中打开：`http://localhost:3000`

## 🎮 开始游戏

### 单人测试
1. 打开浏览器，访问 `http://localhost:3000`
2. 点击"创建房间"
3. 在另一个浏览器标签页（或无痕模式）再次访问 `http://localhost:3000`
4. 点击"加入房间"，输入房间号
5. 房主点击"开始游戏"

### 多人对战
1. 玩家 A：点击"创建房间"，分享房间号
2. 玩家 B：点击"加入房间"，输入房间号
3. 玩家 A 点击"开始游戏"
4. 开始对战！

### 快速匹配
1. 点击"快速匹配"
2. 等待系统自动匹配对手
3. 匹配成功后自动进入游戏

## ⚠️ 常见问题

### Q: 启动失败？
A: 
- 确保已安装 Node.js (版本 14+)
- 检查 3000 端口是否被占用
- 删除 `server/node_modules` 文件夹，重新安装依赖

### Q: 无法连接服务器？
A: 
- 确认服务器正在运行
- 检查浏览器控制台是否有错误
- 尝试刷新页面

### Q: 找不到对手？
A: 
- 使用"创建房间"功能，让朋友加入
- 或在两个浏览器标签页中分别打开游戏进行测试

### Q: 修改端口号？
A: 
编辑 `server/server.js`，修改这一行：
```javascript
const PORT = process.env.PORT || 3000; // 改成你想要的端口
```

## 📱 移动端测试

### 在局域网内访问
1. 查看你的本地 IP 地址
   - Windows: 运行 `ipconfig`
   - Mac/Linux: 运行 `ifconfig`
2. 在手机浏览器访问 `http://你的IP地址:3000`
3. 例如：`http://192.168.1.100:3000`
